# SecureZip 기획서

## 개요

설치 없이 브라우저에서 바로 사용하는 암호 ZIP 압축/해제 도구.
모든 처리가 클라이언트(브라우저)에서 이루어지며 파일이 서버로 전송되지 않는다.

## 목적

- 비개발자도 쉽게 암호 ZIP을 만들고 풀 수 있는 도구 제공
- 설치 없이 iPhone/iPad/PC 어디서든 접근 가능
- 파일이 외부로 유출되지 않는다는 신뢰 제공

## 핵심 기능

### 압축 탭
| 기능 | 설명 |
|------|------|
| 다중 파일 선택 | 드래그앤드롭 또는 클릭 선택 |
| 파일 순서 변경 | 드래그앤드롭 (데스크톱 + 모바일 터치) |
| 파일명 설정 | 커스텀 파일명, 기본값 `archive` |
| 암호 설정 | 선택 사항, AES-256 암호화 |
| 비밀번호 강도 표시 | 4단계 강도 바 |
| 진행률 표시 | 파일별 진행 상태 |
| 자동 다운로드 | 완료 시 즉시 다운로드 |

### 압축 해제 탭
| 기능 | 설명 |
|------|------|
| ZIP 파일 선택 | 드래그앤드롭 또는 클릭 |
| 파일 목록 미리보기 | ZIP 내부 파일명 + 크기 표시 |
| 암호 입력 | 암호 없는 ZIP도 지원 |
| 다운로드 방식 | 파일 1개 → 직접 / 여러 개 → `_extracted.zip` |

## 디자인 시스템

**Obsidian Prism** 다크 테마

| 항목 | 값 |
|------|----|
| 배경 | `#0a0a0a` |
| 포인트 컬러 | `#6366f1` (Indigo) → `#8b5cf6` (Purple) 그라디언트 |
| 액센트 | `#E4F222` (Yellow-Green) |
| 본문 폰트 | Inter |
| 코드 폰트 | JetBrains Mono |

## 다국어

`navigator.language`를 감지해 자동 전환.

- `ko`로 시작하면 → 한국어
- 그 외 → 영어

## 기술 스택

| 항목 | 선택 |
|------|------|
| 구현 | 순수 HTML/CSS/JS 단일 파일 |
| ZIP 처리 | @zip.js/zip.js v2.7.52 |
| 암호화 | AES-256 (zip.js 내장) |
| 배포 | Docker (nginx:alpine) + kamal-proxy |
| 인프라 | Hetzner CPX22 (싱가포르) |
| DNS | Cloudflare (`securezip.sagevee.com`) |
